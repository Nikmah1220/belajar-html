Penjelasan:
<!DOCTYPE html> untuk menentukan tipe dokumen
<HTML> … </HTML> kumpulan karakter yang mengawali dan mengakhiri suatu tag. Karakter yang digunakan (<>). Tag awal disebut opening tag, tag akhir disebut ending tag ditandai dengan (/)
 <HEAD> … </HEAD> Diantaranya adalah area dari heading halaman. Digunakan untuk script/perintah khusus yang tidak mempunyai hubungan dengan format dari halaman. 
<TITLE> … </TITLE> Digunakan untuk pemberian judul
<body></body> Tempat dibuatnya semua konten website menggunakan HTML
<button></button> ini buat tombol
<p></p> ini buat tag paragraf
<h1> </h1> Membuat judul atau heading
</div> untuk mengelompokkan elemen atau tag-tag agar menjadi suatu grup. Tag div ini juga sering digunakan untuk mendefinisikan id atau class dari css
